Version 2.3

May 10, 2023

New content

  • Added Lass(T) hero
  • While adding new hero to Planning tab, You can now specify whether You want to open respective core and input how many Hero specific SI cubes You have
  • Added available energy packages for quick select while configuring Energy income from packages on Energy Calculator page

Bugfixes

  • Fixed bug, where units with mixed sets would only appear when 'All' filter was enabled,
  • Fixed Energy Calculator producing wrong values due to last patch adjustments,

QoL changes

  • Heroes on Tier List tab are now sorted by Cores open before sorted alphabetically
  • Equipment sets colors are changed to match in-game colors
  • Energy Calculator page now shows additionally how many weeks You need to get 3000 SI cubes (new SR char), 3250 (new T char) and 3150 (new T char with cubes from event)
