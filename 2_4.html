Version 2.4

May 12, 2023

New content

  • Added profile management. You can now add / edit / import / switch on the fly between profiles, for example if You have few accounts
  • To avoid any data loss, please make a backup of Your profile.xml file before You proceed with update!

Bugfixes

  • Profile data is now stored in AppData folder. This should resolve issues with progress not saving when app was installed in C:/Program Files
  • Fixed rare bug, that on some machines few hero images were not showing

QoL changes

  • Profile data now saved as json file instead of xml
  • While editing hero equipment tier, option 'None' is now at the top of the list
  • Added energy from gems to predefined energy packages on Energy Calculator page
  • Changing any values in Inventory and then changing tab to Planning now recalculates days needed to finish heroes
