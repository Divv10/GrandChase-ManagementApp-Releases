Version 3.0

July 19, 2023

New features

  • Added localization. For now You can change between English and Portuguese Languages. If You want to help with translations to other languages let me know!
  • Multi edit. You can now quickly edit various stats for many heroes at once,
  • Super compact view. You can now display all of Your heroes with most of the stats entered on one screen,
  • Gear slots. Added tab where You can enter Your gear slots upgrade progress with very approximate calculations on how much mats You need to max it all.

Bug Fixes

  • Fixed minimum / maximum amount of SI traits opened showing wrong values when opening hero edit,
  • Added T3 to accessory tab filters,
  • Changed Accessory editor on Accessories page, so the cards wont resize anymore and come back displaced,
  • Various fixes of wrong calculations on planning tab.

QoL changes

  • Logs will now be saved to AppData folder instead of installation folder.

Installer changes

  • Installer now creates shortcut on desktop,
  • Starting this version, installer now remembers last installation folder and suggests it upon new version installation (so the new version after 3.0 will suggest same path as version 3.0 was installed to).
